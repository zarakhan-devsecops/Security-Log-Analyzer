# Security-Log-Analyzer
Python-based cybersecurity project that analyzes authentication logs and identifies suspicious activity.
## Features
- Detects failed login attempts
- Identifies brute-force attacks
- Tracks suspicious IP addresses
- Generates security alerts

## Technologies
- Python
- Log Analysis
- Security Monitoring
